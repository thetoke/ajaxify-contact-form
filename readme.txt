Ajax Contact Form

== Description ==

Contact Form is one of the most important page for all websites/blogs.
If you’re using Wordpress CMS, than it is one of the easiest things to do.
This is an elegant solution that puts a “sliding contact panel” on your Wordpress site.
Contact form can complement just about any magento installation in a good way.
Its simple and straightforward layout makes for an easy way for the visitor to get in touch with you.

== Installation ==

Upload the AJAX Contact Form plugin to your blog. Activate it, DONE!

1, 2, 3: You're done!
